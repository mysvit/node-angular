## Update

Install compatible version of NodeJS on local os or docker

**Update packages**
``` bash
npm update
```
globally
```bash
npm update -g
```

**To test the update, run**
``` bash
npm outdated
```
globally
```bash
npm outdated -g --depth=0
```

**To Install latest version**
``` bash
npm install nodemon@lates
```
globally
```bash
npm install npm@latest -g
```
